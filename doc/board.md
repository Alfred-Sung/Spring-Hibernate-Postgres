## Create board
Creates and returns a new Board. Board id is generated by the database and stored.

``
POST /board
``

Name | Type | In | Description
------------ | ------------- | ------------- | -------------
name | String | body | Name of the board

### Code Sample

#### Send
```
POST ../api/v1/board
```
```json
{
  "name": "development"
}
```
#### Response
```
200 OK
```
```json
{
  "id": 1,
  "name": "development",
  "posts": []
}
```
