## Create user
Creates and returns a new user object. User id is generated by the database and stored.

``
POST ../api/v1/user
``

Name | Type | In | Description
------------ | ------------- | ------------- | -------------
first-name | String | body | User's first name
last-name | String | body | User's last name

### Code Sample

#### Send
```
POST ../api/v1/user
```
```json
{
  "first-name": "first",
  "last-name": "last"
}
```
#### Response
```
200 OK
```
```json
{
  "id": 4,
  "first-name": "first",
  "last-name": "last",
  "posts": []
}
```
