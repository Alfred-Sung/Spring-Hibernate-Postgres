## Create user
Creates and returns a new user object. User id is generated by the database and stored.

``
POST /user
``

### Code Sample

#### Send
```
POST ../api/v1/user
```
```json
{
    "first-name": "John",
    "last-name": "Smith"
}
```
#### Response
```
200 OK
```
```json
{
    "first-name": "John",
    "last-name": "Smith",
    "id": "fdI4QYYBS4apesftdm5dcg"
}
```
