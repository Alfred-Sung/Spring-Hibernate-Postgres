## Create post
Creates and returns a new post object. File id is generated by the database and stored.

``
POST /post/{userId}/{boardId}
``

Name | Type | In | Description
------------ | ------------- | ------------- | -------------
userId | String | path |
boardId | String | path |
text | String | body |

### Code Sample

#### Send
```
POST ../api/v1/post/1/0
```
```json
{
  "text": "new post"
}
```
#### Response
```
200 OK
```
```json
{
  "id": 4,
  "date-created": "2021-05-16T15:10:42.017343",
  "date-modified": "2021-05-16T15:10:42.017343",
  "text": "new post"
}
```
